显卡：Rx560

BIOS设置

BIOS版本：F8
Advanced CPU Settings → VT-d → Disabled
Extreme Memory Profile (X.M.P.) → Profile 1
Internal Graphics → Enabled
Above 4G Decoding → Enabled
XHCI Handoff → Enabled
Windows 8/10 Features → Other OS
CSM Support → Disabled
