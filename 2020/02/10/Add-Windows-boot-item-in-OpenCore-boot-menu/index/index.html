在OpenCore引导菜单添加Windows引导项

OpenCore配置文件修改

在Misc-HideSelf去掉勾选

在Misc-BlessOverride添加\EFI\Microsoft\Boot\bootmgfw.efi，如下图：

添加Windows启动文件

在Windows的引导ESP里面提取引导文件夹包含Microsoft\Boot\bootmgfw.efi放入OpenCore的EFI文件夹里面。

完成后引导菜单如下图：

可以正常引导Windows！！
