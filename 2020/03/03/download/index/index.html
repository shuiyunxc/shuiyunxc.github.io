如何快速从GitHub下载文件

众所周知，从GitHub下载文件，是一件很痛苦的事情，即便是用GitHub的客户端下载，也是龟速，还经常丢包导致下载失败，这里介绍是利用码云（https://gitee.com）从github克隆项目，然后在码云下载，速度飞快！！当然还有其他梯子办法。

首先你要有码云账户：

没有的话自己注册个，注册简单快速，反正以后会经常用到。

码云注册完成后具体步骤：

打开码云新建仓库

名字随便或与github项目同名

选择导入已有仓库，将GitHub仓库的链接复制

将上面复制的链接中输入导入窗口（如：https://github.com/xxxxx/xxxxx.git）

然后点击创建

等待码云克隆项目，大概1-3分钟（由克隆项目大小决定），完成克隆。

然后再下载码云上项目，速度飞快！！

最后下载完成后不需要这个项目了可以在码云上去删了，也可以保留，以后有更新同步一下，就可以得到更新后的项目。

另外，还有一种更直接快速下载的方法

终端输入：git clone https://gitclone.com/github项目地址

例如：git clone https://gitclone.com/github.com/acidanthera/OpenCorePkg.git
